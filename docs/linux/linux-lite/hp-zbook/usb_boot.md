# Завантажитись з флешки на HP ZBook

## Щоб завантажити HP ZBook з флешки, виконай такі кроки:

   - Встав флешку у ноутбук у будь-який USB-порт.
   - Увімкни ноутбук.
   - Одразу кілька разів натискай **Esc**.
   - З'явиться стартове меню HP.
   - У стартовому меню обери **Boot Device Options (F9)**.
   Там має з’явитися твоя флешка — зазвичай як:
      * USB Hard Drive
      * USB Disk
      * або назва виробника флешки
   - Обери флешку у списку й натисни Enter.

## Якщо флешки в списку немає, перевір у BIOS:

   - Натисни **F10 (BIOS Setup)**.
   - Перейди в **System Configuration → Boot Options**.
   - Увімкни:
     * **Legacy Support** (якщо встановлюєш старі ОС)
     * **UEFI Boot** (сучасні системи)
     * **USB Boot** має бути Enabled
   - Якщо ввімкнений **Secure Boot**, інколи треба вимкнути (**Secure Boot → Disable**), щоб флешка визначилася.
   - Збережи налаштування (**F10 → Yes**) і знову зайди в **F9 Boot Menu**, вибери флешку.
