# Затримка при завантаженні системи


## Це вилазило на більшості компів і ноутів:

`A start job is running for Raise network interfaces...`

Якщо ваша система не залежить від мережі під час завантаження, ви можете вимкнути очікування конфігурації мережі, змінивши установки служби:

```bash
sudo systemctl disable networking.service
```


## На ноуті HP

... побачив, що вісить на такому: `NetworkManager-wait-online.service`

Найпростіше і безпечне рішення — **повністю відключити очікування мережі під час завантаження**, бо гарантія, що мережа піднята до старту всіх сервісів, для звичайного ноутбука, рідко потрібна.


```bash
sudo systemctl disable NetworkManager-wait-online.service
sudo systemctl mask NetworkManager-wait-online.service
```

* `disable` — прибирає автозапуск при старті.
* `mask` — повністю блокує запуск сервісу навіть випадково.

Після цього ноутбук завантажуватиметься без затримки, а NetworkManager все одно підніме мережу на звичайний режим.

!!! note

    Коли це зробив на HP-ноуті, після цього з'явився перший варіант затримки. Заблокував ще й networking.service. Мовби все працює.
