# MariaDB


**MariaDB** — це реляційна система керування базами даних, створена у 2009 році як відкрита та сумісна з MySQL. Причиною створення MariaDB стало занепокоєння спільноти та частини розробників MySQL після того, як Sun Microsystems, власник MySQL, була придбана Oracle у 2009 році. Було побоювання, що майбутній розвиток MySQL стане менш відкритим, темп релізів сповільниться або продукт почне орієнтуватися насамперед на комерційні інтереси, а не на спільноту. Щоб гарантувати незалежний, прозорий і стабільний розвиток сумісної СУБД, ключові інженери MySQL, включно з Майклом Монті Виденіусом, започаткували MariaDB як відкриту та повністю community-driven альтернативу.

Проєкт зберігає повну сумісність на рівні більшості запитів і структур, тому може слугувати прямою заміною MySQL у багатьох сценаріях без значних змін у застосунках. Система пропонує широкий набір механізмів зберігання, вдосконалену оптимізацію запитів, розширені можливості реплікації й масштабування, а також регулярні відкриті релізи. Завдяки цьому MariaDB увійшла до стандартних репозиторіїв багатьох Linux-дистрибутивів і часто використовується як дефолтний сервер баз даних. Її переваги над MySQL зазвичай проявляються у продуктивності, передбачуваності розвитку, відкритості та розширюваності.
