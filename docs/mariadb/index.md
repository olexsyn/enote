# MariaDB


**MariaDB** — це реляційна система керування базами даних, створена у 2009 році як відкрито розроблювана та сумісна з MySQL альтернатива після зміни власності MySQL. Проєкт зберігає повну сумісність на рівні більшості запитів і структур, тому може слугувати прямою заміною MySQL у багатьох сценаріях без значних змін у застосунках. Система пропонує широкий набір механізмів зберігання, вдосконалену оптимізацію запитів, розширені можливості реплікації й масштабування, а також регулярні відкриті релізи. Завдяки цьому MariaDB увійшла до стандартних репозиторіїв багатьох Linux-дистрибутивів і часто використовується як дефолтний сервер баз даних. Її переваги над MySQL зазвичай проявляються у продуктивності, передбачуваності розвитку, відкритості та розширюваності.
