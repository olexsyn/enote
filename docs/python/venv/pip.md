# pip

pip — це стандартний менеджер пакетів у Python. Тобто маленький інструмент, який дозволяє встановлювати сторонні бібліотеки з Python Package Index (PyPI). Коли ти робиш `pip install requests`, він просто знаходить пакет *requests* у репозиторії, завантажує його і встановлює у твоє середовище.

Без pip довелося б кожну бібліотеку вручну качати, розпаковувати, кидати у правильні каталоги. А pip робить усе сам, ще й залежності підтягне.

Коротко:

* pip = пакетний менеджер Python
* ставить, оновлює, видаляє пакети
* працює з PyPI — головним репозиторієм бібліотек
* використовується практично у всіх проєктах
* інтегрований у Python 3 за замовчуванням
